# Programación 1 — UTN

Trabajos prácticos y ejercicios de **Programación 1** de la Tecnicatura Universitaria en Programación (UTN).

**Lenguaje:** C &nbsp;|&nbsp; **IDE:** Code::Blocks

## Cómo abrir los proyectos

1. Instalar [Code::Blocks](https://www.codeblocks.org/)
2. Abrir el archivo `.cbp` de la carpeta que querés ver
3. Compilar y ejecutar con **F9**
